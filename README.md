# Project X
This is a web project using Tailwind CSS and Express.js.

## Setup Instructions
1. Install dependencies:
   ```sh
   npm install
   ```
2. Run the development server:
   ```sh
   npm run dev
   ```
3. Build Tailwind CSS:
   ```sh
   npm run build:css
   ```
